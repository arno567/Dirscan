# sdirscan 二级目录扫描工具
## 实现功能
批量目标扫描功能：
1、目录x目标进行匹配检测，返回状态码、数据包大小、记录应用ICO logo
2、跳转检测，区分跳转到同一个界面的路径并过滤
3、直接将结果写入xls(未实现)、html
4、自动爬取页面连接，收集二级目录（未实现）

单目标扫描功能：
跟普通扫描工具一样
